Projeto de Monitoramento Ambiental com ESP32

Este projeto foi desenvolvido como parte das atividades da disciplina, criando um sistema inteligente de monitoramento ambiental utilizando microcontroladores.

Objetivos

  • Monitorar temperatura e umidade ambiente em tempo real
  • Controlar automaticamente a iluminação baseado na luminosidade
  • Fornecer alertas visuais quando as condições saírem dos parâmetros ideais
  • Criar uma interface de usuário simples com display OLED

Componentes Utilizados

ESP32

Microcontrolador principal do projeto

DHT22

Sensor de temperatura e umidade

Módulo Relé

Para controle de dispositivos AC

Display OLED

Interface visual 128x64 pixels

Fotoresistor

Sensor de luminosidade

LEDs

Indicadores visuais (vermelho e amarelo)

Diagrama do Circuito

Código Principal

# Importação de bibliotecas
from machine import Pin, ADC, I2C
import time
import dht
import ssd1306

# Configuração do display OLED
i2c = I2C(0, scl=Pin(22), sda=Pin(21))
oled_width = 128
oled_height = 64
oled = ssd1306.SSD1306_I2C(oled_width, oled_height, i2c)

# Inicialização do sensor DHT22
sensor = dht.DHT22(Pin(15))

# Configuração dos LEDs
red = Pin(2, Pin.OUT)
yellow = Pin(4, Pin.OUT)

# Configuração do relé
relay = Pin(23, Pin.OUT)

# Configuração do sensor de luz
AO_PIN = ADC(Pin(32))
AO_PIN.width(ADC.WIDTH_12BIT)
AO_PIN.atten(ADC.ATTN_11DB)

# Mensagem inicial
oled.fill(0)
oled.text('Iniciando...', 25, 25)
oled.show()
time.sleep(2)
relay.value(0)

# Loop principal
while True:
    try:
        sensor.measure()
        temperature = sensor.temperature()
        humidity = sensor.humidity()

        # Controle de iluminação automático
        light_value = AO_PIN.read()
        if light_value >= 2500:
            relay.value(1)
        else:
            relay.value(0)

        # Lógica de alertas
        if humidity > 60:
            if 12 <= temperature <= 38:
                alert = False
                danger = False
            else:
                alert = True 
                danger = False               
        else:
            if 12 <= temperature <= 30:
                alert = True 
                danger = False             
            else:
                danger = True
                alert = False

        # Exibição no display
        oled.fill(0)
        if danger:
            oled.text(f"Perigo! {temperature:.0f}C", 0, 0)
            yellow.off()
            red.on()
        elif alert:
            oled.text(f"Alerta! {temperature:.0f}C", 0, 0)
            yellow.on()
            red.off()
        else:
            oled.text(f"{temperature:.0f}C", 0, 0)
            yellow.off()
            red.off()

        oled.text(f"Umidade: {humidity:.0f}%", 0, 10)

        if relay.value():
            oled.text("Luz ligada", 0, 20)
        else:
            oled.text("Luz desligada", 0, 20) 

        oled.show()

    except OSError as e:
        oled.fill(0)
        oled.text("Erro no sensor", 0, 10)
        oled.text(str(e), 0, 20)
        oled.show()
        
    time.sleep(3)
